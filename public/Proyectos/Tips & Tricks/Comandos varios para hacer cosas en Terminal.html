Convertir archivos

De Word a Markdown

Primero tienes que tener Pandoc instalado: winget install --source winget --exact --id JohnMacFarlane.Pandoc.

Para un solo archivo: pandoc -o "output.md" "input.docx"

De wav a mp3

Tienes que tener FFmpeg instalado: winget install -e --id Gyan.FFmpeg

Para un solo archivo: ffmpeg -i "input.wav" "output.mp3"

Permitir ejecución de scripts en Powershell

Siempre como administrador: Set-ExecutionPolicy unrestricted
