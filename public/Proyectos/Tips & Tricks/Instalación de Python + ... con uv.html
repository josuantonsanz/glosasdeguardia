Instalación de Python con uv

1 min
Plantado hace 3 meses. Atendido hace 3 meses.

Siempre que tengo que volver a instalar Python, el gestor de paquetes y demás acabo buscando en internet, así que lo voy a dejar aquí puesto.

1. Instalar uv (gestor de paquetes)

En terminal, escribir: powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"

2. Instalar python

  • En general: uv python install → Instala el comando python3.13
  • Una versión concreta uv python install 3.12
  • Para instalar python y python3: uv python install --default

3. Instalar Jupyter Notebook

  • En general: uvx jupyter lab
  • Una versión concreta: uvx --python 3.13 jupyter lab
