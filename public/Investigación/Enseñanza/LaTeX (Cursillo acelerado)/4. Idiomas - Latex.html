Idiomas en LaTeX

2 min read

Aunque una vez que funciona es maravilloso, es un poco complicado configurar LaTeX para que funcione con varios idiomas (sobre todo si tienen distinta grafía o dirección). Para los siguientes pasos, se recomienda cambiar el compilador de pdflatex a xelatex. La clase de documento que usaremos será memoir.

Para todos los casos, conviene usar un tipo de letra que tenga todos los caracteres españoles y griegos, así que vamos a usar libertine:

\usapackage{libertine}

Español

Para el español usaremos babel:

\PassOptionsToPackage{spanish, es-lcroman}{babel}
\usepackage{babel}
\selectlanguage{spanish}

Griego

Usando xelatex y libertine como fuente, puedes escribir en griego sin ningún problema (usando un teclado griego). Si no está esa opción, puedes usar textgreek:

\usepackage[euler]{textgreek}

Hebreo

Para el hebreo cjhebrew, con dos comandos extra para simplificar la inclusión de texto hebreo en medio de una línea:

\usepackage{cjhebrew}
\newcommand{\hb}[1]{\cjRL{#1}}
\newcommand{\Lhb}[1]{{\Large\hb{#1}}}

Así, el conjunto quedaría:

\usepackage[euler]{textgreek}
\usepackage{cjhebrew}
\newcommand{\hb}[1]{\cjRL{#1}}
\newcommand{\Lhb}[1]{{\Large\hb{#1}}}
\PassOptionsToPackage{spanish, es-lcroman}{babel}
\usepackage{babel}
\selectlanguage{spanish}

Párrafos completos en hebreo

Aquí se complica un poco la cosa, hay que usar polyglossia y deshacernos de babel. Quedaría tal que así:

\usepackage[euler]{textgreek}
\usepackage{cjhebrew}
\newcommand{\hb}[1]{\cjRL{#1}}
\newcommand{\Lhb}[1]{{\Large\hb{#1}}}

% - Right-to-left text - %

\usepackage{polyglossia}
\let\XeLaTeX\relax
\let\XeTeX\relax
\setmainlanguage{spanish}
\setotherlanguage{hebrew}
\selectlanguage{spanish}

Fonética y transliteración

Para la fonética y transliteración, lo más sencillo es usar tipa. Para más facilidad con el hebreo, podemos añadir estos comandos:

\usepackage{tipa}
\newcommand{\ie}{\textipa{@}} % e invertida (sewa)
\newcommand{\sewa}{\v{s}\ie wa} % transliteración de "sewa"
\renewcommand{\st}{\textprimstress} % acento transliterado

Algunos símbolos comunes: - ò \`{o} - ó\'{o} - ô \^{o} - ǒ\v{o}
- ō \ ={o} - o̲ \b{o} - ȯ \.{o} - ọ \d{o}
- ŏ \u{o} - õ \~{o}
