¿Qué es LaTeX?

LaTeX es un sistema de composición de documentos ampliamente utilizado en entornos académicos y científicos. A diferencia de los procesadores de texto convencionales, LaTeX se basa en el marcado de texto, lo que significa que agregas comandos especiales a tu documento para definir su estructura y formato. Luego, LaTeX procesa estos comandos y genera un documento final con una apariencia profesional y consistente. Es una herramienta esencial para aquellos en el mundo académico que desean producir documentos de alta calidad con facilidad.

Cómo acceder a LaTeX

Para acceder a LaTeX, tienes dos opciones populares. La primera es utilizar Overleaf, una plataforma en línea que te permite crear y editar documentos LaTeX directamente en tu navegador, sin necesidad de instalar software adicional. Es una opción conveniente, especialmente si trabajas en diferentes computadoras o sistemas operativos. La segunda opción es instalar una distribución LaTeX como MikTeX y un editor como TeXstudio en tu ordenador. Esto te proporciona un entorno de LaTeX local, lo que puede ser útil si prefieres trabajar sin conexión a internet o si necesitas un mayor control sobre tu sistema. Ambas opciones son efectivas, y la elección depende de tus preferencias personales y necesidades.

Conceptos básicos

En LaTeX, es fundamental comprender algunos conceptos clave. En primer lugar, LaTeX separa el contenido del documento del formato, lo que significa que puedes enfocarte en escribir tu texto sin preocuparte por el diseño. Esto resulta muy útil, ya que te permite centrarte en la creación de contenido de manera eficiente.

Además, LaTeX funciona mediante un proceso de compilación. En otras palabras, no es un procesador de texto en tiempo real como Word. Necesitas un programa intermedio, llamado compilador, para transformar tus archivos de texto con formato LaTeX en documentos finales. Esto asegura que el formato sea coherente y profesional en todo momento.

Es importante destacar que LaTeX, en su estado inicial, proporciona solo las funcionalidades básicas. Sin embargo, puedes personalizarlo a tus necesidades mediante el uso de "paquetes". Estos son complementos que añaden funciones adicionales, como fuentes específicas o estilos de página personalizados, permitiéndote adaptar LaTeX para satisfacer tus requerimientos particulares.

Domando a LaTeX: Comandos y formato

Unidades básicas de actuación: comandos y entornos

En LaTeX, un comando es una instrucción especial que empieza con una barra invertida, como \comando. Algunos comandos tienen argumentos, que son información adicional que le dices al comando para que sepa qué hacer. Por ejemplo, el comando \section se usa para crear una sección en tu documento y el argumento es el título de la sección, como \section{Introducción}. Además, algunos comandos pueden tener argumentos opcionales entre corchetes, como \comando{argumento}[opcion]. Estos comandos son fundamentales para dar formato y estructura a tu documento de manera precisa.

En LaTeX, un entorno o grupo es una forma de delimitar un conjunto de comandos que afectan a un bloque específico de texto. Se inicia con \\begin{entorno} y se cierra con \end{entorno}. Por ejemplo, \begin{quote} y \end{quote} crean un entorno que formatea el texto como una cita. Otro ejemplo es el entorno itemize, que se usa para crear listas con viñetas, comenzando con \begin{itemize} y terminando con \end{itemize}. Los entornos son útiles para aplicar formato a secciones específicas de tu documento.

Estructura de un documento de LaTeX

En LaTeX, la estructura de un documento sigue un proceso sencillo pero organizado.

Cabecera: Comienza por elegir el tipo de documento que deseas crear, lo que es como seleccionar una plantilla predefinida. Esto establece las reglas básicas de formato para tu documento. Todo documento comienza con \documentclass[opciones]{clase de documento}. Hay muchas clases de documento entre las que elegir, pero estas son las principales: 1. article: Para documentos tipo artículo, como informes, ensayos o trabajos académicos. 2. report: Para documentos más largos, como informes extensos o tesis. 3. book: Para la creación de libros y documentos más largos que se dividen en capítulos. 4. letter: Para escribir cartas. 5. beamer: Para la creación de presentaciones.

Además, estás son las opciones comunes: 1. 10pt, 11pt, 12pt: Establecen el tamaño de fuente del documento a 10, 11 o 12 puntos, respectivamente. 2. a4paper, letterpaper: Define el tamaño del papel del documento como A4 o carta (tamaño estándar de EE. UU.). 3. twocolumn: Formato de dos columnas en lugar de una. 4. titlepage, notitlepage: Indica si se debe crear una página de título o no. 5. twoside, oneside: Configura el documento para impresión a doble cara o a una sola cara. 6. openright, openany: Controla si los capítulos comienzan en una página derecha o en cualquier página. 7. landscape: Cambia la orientación de la página a apaisada (horizontal).

Preámbulo: Aquí es donde defines todas las configuraciones y ajustes especiales que deseas aplicar a tu documento. Esto incluye la inclusión de paquetes adicionales (como mencionamos antes) y la definición de elementos como el título, el autor y otros detalles de formato. Esta parte es como establecer las reglas específicas de tu documento.

Documento: Finalmente, está el «contenido». Esta es la parte principal de tu documento, donde escribirás el texto y agregas elementos como títulos, secciones, tablas, imágenes y ecuaciones. LaTeX se encargará de formatear todo según las reglas que estableciste en el preámbulo. 

\begin{document}
    Aquí va el contenido
\end{document}

Esta estructura te brinda un alto grado de control sobre el formato y la organización de tus documentos.

Título, autor y fecha

El título, autor y fecha en un documento LaTeX son elementos metadatos que se utilizan para dar información básica sobre el documento. El comando \title{} se usa para definir el título del documento, que generalmente aparece en la página de título. \author{} establece el nombre del autor del documento. Por último, \date{} se utiliza para especificar la fecha, que puede ser la fecha actual, una fecha personalizada o estar en blanco. Una vez que se han definido estos elementos, se utiliza \maketitle para generar la página de título con la información proporcionada. Esto asegura que el título, autor y fecha se muestren de manera uniforme y profesional en todo el documento. Aquí tienes un ejemplo:

\documentclass{article} 

\title{Mi Primer Documento en LaTeX} 
\author{Josu} 
\date{\today} 

\begin{document} 

\maketitle 

Este es un ejemplo de documento en LaTeX que incluye el título, autor y fecha.

\end{document}

Espacios y párrafos

  1. Espacios:
    • Para agregar un espacio en blanco simple entre palabras, simplemente escribe un espacio en el texto.
    • LaTeX no tiene en cuenta múltiples espacios o líneas en blanco seguidas. Si necesitas espacios adicionales, puedes usar \quad para un espacio grande o \hspace{1em} , poniendo en 1em el tamaño que necesites
    • Para crear un espacio no rompible, utiliza \~{} en lugar de un espacio normal. Esto evitará que LaTeX lo divida al final de una línea.
  2. Nueva Línea:
    • Para comenzar una nueva línea sin comenzar un nuevo párrafo, simplemente usa \\ al final de una línea.
  3. Nuevo Párrafo:
    • Para comenzar un nuevo párrafo, simplemente deja una línea en blanco en tu código fuente. LaTeX entenderá que estás iniciando un nuevo párrafo.

Capítulos y secciones

En LaTeX, la organización de documentos se logra mediante una jerarquía que incluye «partes» (\part{}), «capítulos» (\chapter{}), «secciones» (\section{}), y "subsecciones" (\subsection{}). Cada uno de estos elementos se utiliza para estructurar el contenido de manera organizada y lógica, permitiendo una fácil navegación a través del documento. Las partes son la división más alta, seguidas de los capítulos, secciones y subsecciones, lo que proporciona una jerarquía que se refleja en la numeración automática de estas divisiones en el documento final.

Dando formato

Tamaño de letra

  1. \tiny{Texto}: Texto muy pequeño.
  2. \scriptsize{Texto}: Texto un poco más grande que \tiny.
  3. \footnotesize{Texto}: Tamaño de fuente de nota a pie de página.
  4. \small{Texto}: Tamaño de fuente pequeño.
  5. \normalsize{Texto}: Tamaño de fuente normal.
  6. \large{Texto}: Tamaño de fuente grande.
  7. \Large{Texto}: Tamaño de fuente aún más grande.
  8. \LARGE{Texto}: Texto muy grande.
  9. \huge{Texto}: Texto extremadamente grande.
  10. \Huge{Texto}: Tamaño de fuente máximo.

Negrita, cursiva y subrayado

  1. \textbf{Texto}: Para texto en negrita.
  2. \textit{Texto}: Para texto en cursiva.
  3. \underline{Texto}: Para subrayar texto.
  4. \texttt{Texto}: Para texto de tipo de máquina.
  5. \textsc{Texto}: Para texto en versalitas.

Listas

En LaTeX, puedes crear listas de diferentes tipos: numeradas, con viñetas o de descripción. Aquí te explico cómo hacerlo:

Lista Numerada: Para crear una lista numerada, utiliza el entorno enumerate. Aquí tienes un ejemplo:

\begin{enumerate}     
    \item Primer elemento.     
    \item Segundo elemento.     
    \item Tercer elemento. 
\end{enumerate}

Esto generará una lista con números, como:

  1. Primer elemento.
  2. Segundo elemento.
  3. Tercer elemento.

Lista con Viñetas: Para crear una lista con viñetas, utiliza el entorno itemize. Aquí tienes un ejemplo:

\begin{itemize}     
    \item Elemento con viñeta.     
    \item Otro elemento con viñeta.     
    \item Y otro más. 
\end{itemize}

Esto generará una lista con viñetas, como:

  • Elemento con viñeta.
  • Otro elemento con viñeta.
  • Y otro más.

Lista de Descripción: Para crear una lista de descripción, utiliza el entorno description. Aquí tienes un ejemplo:

\begin{description}     
    \item[Etiqueta 1] Descripción del primer elemento.     
    \item[Etiqueta 2] Descripción del segundo elemento.     
    \item[Etiqueta 3] Descripción del tercer elemento. 
\end{description}

Esto generará una lista en la que puedes etiquetar cada elemento con una descripción, como:

Etiqueta 1: Descripción del primer elemento. Etiqueta 2: Descripción del segundo elemento. Etiqueta 3: Descripción del tercer elemento.

Estos entornos de lista son muy útiles para organizar la información en tu documento de una manera clara y ordenada. Puedes personalizar aún más el formato de estas listas si es necesario.

Un ejemplo completo

\documentclass{article} 

\title{Mi Primer Documento en LaTeX} 
\author{Josu} 
\date{\today} 

\begin{document} 
    \maketitle 

    \section{Introducción}

     \subsection{Subsección} 

     Aquí tienes una subsección dentro del cuerpo del documento. Tiene \textbf{texto en negrita} o \huge{muy grande}.
