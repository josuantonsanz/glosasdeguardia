Cómo crear una instalación de Python sencilla en Windows

En realidad, es fácil.

1. Instalar Python

A poder ser, no la última versión, sino una que sea compatible con casi todo. En mi caso, Python 3.12 —la anterior a la última, 3.13— desde python.org. Es importante, al instalarlo, asegurarse que se le da al botón de «Dissable path length limit».

También viene bien instalar un gestor de paquetes; preferiblemente, uv. Se puede usar el comando:

powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"

2. Instalar algún IDE

Se puede usar VSCode, o el IDLE de Python. Yo prefiero, PyCharm. Por todo lo que integra desde el principio.
